# Triage Labels

Local Markdown work items may use these canonical states:

| Workflow role | Local label |
| --- | --- |
| Unclassified | `needs-triage` |
| Missing a required decision | `needs-info` |
| Fully specified for implementation | `ready-for-agent` |
| Requires human authority or external action | `ready-for-human` |
| Deliberately rejected | `wontfix` |

The canonical research-flywheel initiative is `ready-for-agent` unless its progress ledger records a blocking human decision.
